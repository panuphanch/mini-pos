## Thai Font for Receipt Printing

`NotoSansThai-Regular.ttf` (Noto Sans Thai variable font, OFL) is committed to
this directory and embedded into the binary at compile time via
`include_bytes!`. No runtime file lookup is performed.

To update the font, replace the file with a newer release from
https://fonts.google.com/noto/specimen/Noto+Sans+Thai and rebuild.
